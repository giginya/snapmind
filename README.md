# SnapMind

SnapMind is a mobile-friendly app that converts screenshots into usable text-based knowledge.

## Features

- Upload screenshots
- Extract text using OCR
- Clean and structure notes
- Save and view notes

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
